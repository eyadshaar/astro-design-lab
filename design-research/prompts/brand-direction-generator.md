# Brand Direction Generator
## For when a client has no existing brand direction

---

## Prompt

You are a brand strategist. Generate a distinctive brand direction for the following business.

**Business:** [Name]
**Industry:** [Industry]
**Product/Service:** [What they sell]
**Target Audience:** [Who buys this]
**Primary Website Goal:** [What the website needs to accomplish]

**Design Reference Sites (from research catalog):**
- Navbar Gallery (navbar.gallery) — navigation patterns
- CTA Gallery (cta.gallery) — CTA and conversion patterns
- Landing.love (landing.love) — animation and landing page patterns
- Mobbin (mobbin.com) — mobile and app patterns
- Rebrand.gallery (rebrand.gallery) — brand identity patterns

**Task:**

Generate a complete brand direction document with:

### 1. Visual Metaphor
What is the one visual concept that ties this brand together? Name it specifically. Not "clean modern" but "industrial workshop," "command console," "curated journal," etc.

### 2. Color Territory
What color family is unexpected for this industry? Everyone in [industry] uses [color]. What territory is unclaimed?

- Primary: [color + hex]
- Secondary: [color + hex]
- Accent: [color + hex]
- Background: [color + hex]
- Text: [color + hex]

Justify each choice in one sentence.

### 3. Typography Direction
What typeface personality fits this brand?

- Display (headlines): [Typeface name + Google Fonts link]
- Body: [Typeface name + Google Fonts link]
- Mono (technical): [Typeface name + Google Fonts link]

Why does this pairing work for [industry]?

### 4. Layout Strategy
What is the dominant layout pattern? How does it support the visual metaphor?

### 5. Motion Temperament
How does animation feel? Fast and precise? Slow and cinematic? Functional only?

### 6. One Unexpected Element
What is the one design element that would make this site memorable — a specific section motif, interaction pattern, or visual device — that most competitors don't use?

## Example Output

**Business:** Local Portland coffee shop
**Industry:** Specialty coffee / cafe
**Audience:** 25-40 professionals who work remotely, want good coffee and a place to focus

### 1. Visual Metaphor
**The Day-Starts-Here Journal.** The site feels like a well-worn field notebook — coffee-stained edges, handwritten annotations, the smell of ink and beans. The homepage is a spread from the owner's brew journal.

### 2. Color Territory
Most coffee sites use brown/warm neutrals. This shop owns **cool slate + amber**:

- Primary: #2C2C2C (rich charcoal — grounds the brand)
- Secondary: #D4A574 (warm amber — the coffee itself)
- Accent: #8B9A82 (muted sage — the plants they serve alongside)
- Background: #FAFAF8 (warm off-white — like cream)
- Text: #1A1A1A (near-black)

### 3. Typography Direction
- Display: **Playfair Display** — editorial, classic, a touch of luxury
- Body: **DM Sans** — clean, modern, highly legible
- Mono: **JetBrains Mono** — for menu item codes

The serif/sans contrast matches "craft coffee shop that uses modern POS."

### 4. Layout Strategy
Magazine editorial layout. One dominant image per section, text in narrow columns. The menu is presented like a journal entry — handwritten-style headers with organized item lists. Not a grid of items.

### 5. Motion Temperament
Slow, warm fades. Nothing bounces. The pace of morning coffee — unhurried. Text appears like pages turning.

### 6. One Unexpected Element
The menu is presented as a physical menu card photographed at the table — with coffee ring stains and handwritten annotations. No other coffee shop site looks like this.
